# gspbs-web-app

## Specs
- PHP web based application.
- Using a Google Sheet API to store details.

## Modules
1. Login
    - User Login
    - Admin Login
    
2. Admin
    - Approve/Deny any registered a USER.
    - Post updates/news.
    - Search filter for phone directory.
    
3. User
    - Search filter for committe members.
    - Search filter for phone directory. (Hide Contact Details)
